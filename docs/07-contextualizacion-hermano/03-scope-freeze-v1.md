# 03 - Scope Freeze V1 (version operativa)

Fuente: `docs/00-contexto/01-scope-freeze-v1.md`

## Incluido en V1

- Web institucional responsive con secciones principales.
- Reservas con calendario + formulario + confirmacion.
- Integraciones: Supabase, HubSpot, Cal.com, WhatsApp y Vercel.
- Cursos/inscripciones basicas y segmentacion inicial.
- Email marketing inicial y automatizaciones base.
- SEO basico, Google Maps, GA4 y Search Console.

## No incluido en V1

- LMS completo (aula virtual avanzada, progreso, foros, certificados).
- Reescrituras de arquitectura por preferencia.
- Integraciones o automatizaciones avanzadas no listadas.
- Apps moviles nativas.

## Regla de cambios

Todo pedido fuera de este alcance se registra como Change Request con aprobacion y cotizacion independiente.

## SLA de soporte de referencia

- Respuesta inicial: hasta 48 horas habiles.
- Incidencia critica: hasta 24 horas habiles.
- Consultas generales: hasta 72 horas habiles.
- Correcciones/ajustes: hasta 2 semanas calendario.
